# Event Management API

This project is a RESTful Web API built using **FastAPI** and **MongoDB** for managing event-related data.

## Features
- Manage events, attendees, venues, and ticket bookings
- Upload and retrieve multimedia files (event posters, promotional videos, venue photos)
- Secure database access using MongoDB Atlas

## Technologies Used
- Python 3
- FastAPI
- Uvicorn
- MongoDB Atlas
- Motor (Async MongoDB driver)
